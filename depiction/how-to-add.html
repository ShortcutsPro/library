How to add a repo to Papercuts

By eaic

What is a repo?

In Papercuts, a repo (or source), is a collection of Shortcuts. To use a repo, add it to a Papercuts client (an app, website, or Shortcut that uses Papercuts technology).

How do I add a repo?

To add a repo to the Papercuts web client, open it
Click "Sources"

Click the add button

Paste the repo link

Click "Add"
